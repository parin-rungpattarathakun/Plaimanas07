Plaimanas07 is a Thai-Latin typeface project developed by Cadson Demak, with the intention of exploring new possibilities in Thai type design. This project serves both as a developmental prototype and as a platform for educational and collaborative use.

Note: Plaimanas07 is an original design.
All glyphs and outlines have been created from scratch without relying on existing typefaces.

To follow the development or contribute to the project, please visit:
https://github.com/parin-rungpattarathakun/Plaimanas07
